# Cluster Service

Global cluster map service for Cluster.

## Features
- Global cluster configuration
- Tribe aggregation
- Two view modes (global + circle)

## Endpoints
- `GET /api/cluster/config` - Get cluster config
- `GET /api/cluster/global-view` - Get global view (protected)

## Setup
1. Install dependencies: `npm install`
2. Configure .env file
3. Run: `npm run dev`

Port: 3006
